৪/async/await মুলত এসিনক্রোনাস ডেটা নিয়ে কাজ করে। এপি আই থেকে ডেটা এসিনক্রোনাস ভাবে লোড করে এবং এরর হ্যান্ডেলিং ও সহজ করে। await শুধূ এসিনক্রোনাস ফাংশান এই ব্যাবহার করা যায়।

৫/ Scope বলতে বোঝায় কোন ভ্যারিয়েবল বা ফাংশন কোথায় ব্যবহার করা যাবে । মূলত তিন ধরনের Scope আছে।  Global Scope মানে সরজায়গায় পাওয়া যায়, Function Scope মানে শুধু ফাংশনের ভেতরে পাওয়া যায়, আর Block Scope মানে শুধু ব্লকের ভেতরে পাওয়া যায়
